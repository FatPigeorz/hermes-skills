# Create Project

Use this skill when FatPigeorz asks Hermes/Tom to create a new research project.

## Required naming

1. Convert the project name into a lowercase slug.
2. Prefix Discord channel names with `proj-`.
   - Example: `proj-agent-safety`
3. Prefer the same slug for the GitHub repository unless the user requests otherwise.

## Notion workspace structure

The Notion integration currently sees the research project workspace as:

- Root page: `Research Projects`
  - Page ID: `34bb24bd-0bb7-80ff-8f94-ccd891aa66e8`
  - URL: `https://www.notion.so/Research-Projects-34bb24bd0bb780ff8f94ccd891aa66e8`
- Existing example project page: `test`
  - Child pages: `Literature`, `Experiments`, `Methods`
  - The `Experiments` child page contains an inline database named `Experiments`.
  - Experiments data source ID: `4d03eb69-e250-4c38-a31d-0849704ac252`
  - Experiment properties: `Name` (title), `Motivation` (rich_text), `Finding` (rich_text).

## Workflow

1. Create a channel in the Jerry&Tom Discord server under the `projects` category.
   - Channel name format: `proj-xxx` using lowercase.
   - Verify the channel is created in the correct category.
2. Create the related Notion project page under the `Research Projects` root page.
   - Use project page title matching the project name/slug.
   - Add child pages: `Literature`, `Experiments`, and `Methods`.
   - Add or maintain an `Experiments` database with at least `Name`, `Motivation`, and `Finding` properties.
   - Link the Discord channel and GitHub repository in the page when available.
3. Create a private repository in the `Agentiix` GitHub organization.
   - Organization URL: `https://github.com/Agentiix`
   - Repository visibility: private.
4. Initialize the repository locally with `uv init`.
   - Create or clone the repo locally first.
   - Run `uv init` in the repository root.
   - Commit the initialized project files.
   - Push the initial commit to the private `Agentiix/<repo>` repository.

## Verification checklist

Before reporting success, verify:

- Discord channel exists and is under the `projects` category.
- Notion page exists and is reachable by the integration.
- GitHub repository exists under `Agentiix` and is private.
- Local repository contains the files generated by `uv init`.
- Initial commit is pushed to GitHub.

## Output format

```text
Project created: <name>
Discord: <channel mention/link>
Notion: <page URL>
GitHub: https://github.com/Agentiix/<repo>
Local path: <path>
Verification: <brief checklist>
Missing/blocked: <credentials, permissions, parent Notion location, etc.>
```
